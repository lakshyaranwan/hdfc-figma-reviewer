# Design Analyzer - Figma Plugin

A Figma plugin that analyzes your designs for consistency, UX, accessibility, and more.

## Installation

1. **Install dependencies:**
   ```bash
   cd figma-plugin
   npm install --save-dev @figma/plugin-typings typescript
   ```

2. **Compile TypeScript:**
   ```bash
   npx tsc
   ```

3. **Load in Figma:**
   - Open Figma Desktop
   - Go to **Plugins** → **Development** → **Import plugin from manifest...**
   - Select the `manifest.json` file from this folder

## Usage

1. Open any Figma file
2. Run the plugin: **Plugins** → **Development** → **Design Analyzer**
3. Enter your Figma API key (get one from figma.com/developers)
4. Select analysis categories or write a custom prompt
5. Click **Analyze Design**
6. Review feedback and optionally post comments to Figma

## Features

- **Category-based analysis**: Consistency, UX, Accessibility, Visual, Layout, Typography
- **Custom prompts**: Write your own analysis criteria
- **Selection-aware**: Analyzes selected elements or full page
- **Comment posting**: Post feedback directly as Figma comments
- **Severity indicators**: High, Medium, Low priority issues

## Files

- `manifest.json` - Plugin configuration
- `code.ts` - Main plugin code (runs in Figma sandbox)
- `code.js` - Compiled JavaScript (generated by tsc)
- `ui.html` - Plugin UI

## API Key

You need a Figma Personal Access Token:
1. Go to https://www.figma.com/developers
2. Click "Get Personal Access Token"
3. Copy and paste into the plugin

## Development

To make changes:
1. Edit `code.ts` or `ui.html`
2. Run `npx tsc` to recompile
3. In Figma, close and reopen the plugin to see changes
